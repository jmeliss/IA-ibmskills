DADOS NÃO ESTRUTURADOS

Dados são informações brutas. Os dados podem ser fatos, estatísticas, opiniões ou qualquer tipo de conteúdo 
registrado em algum formato. 
Isso pode incluir vozes, fotos, nomes e até movimentos de dança!

Dados estruturados normalmente são classificados como dados quantitativos e são altamente organizados. 
Dados estruturados são informações que podem ser organizadas em linhas e colunas. 

Dados não estruturados, também conhecidos como dados obscuros, são normalmente categorizados como dados 
qualitativos. . Eles não podem ser processados e analisados por ferramentas e métodos de dados convencionais,
eles precisam de organização ou estrutura integrada para serem analisados.

Dados semiestruturados são a "ponte" entre dados estruturados e dados não estruturados. 
Eles não têm um modelo de dados predefinido, são mais complexos do que os dados estruturados, mas mais fáceis 
de armazenar do que os dados não estruturados, eles usam metadados para identificar caracteristicas específicas
de dados e escalar em registors e campos pré-configurado. Utilizam hashtag para identificar uma localização.

Hoje, cerca de 80% dos dados do mundo não são estruturados.

O QUE É APRENDIZADO DE MÁQUINA
O aprendizado de máquina (em inglês, machine learning) é um método de análise de dados que automatiza a 
construção de modelos analíticos. É um ramo da inteligência artificial baseado na ideia de que sistemas podem 
aprender com dados, identificar padrões e tomar decisões com o mínimo de intervenção humana.

O aprendizado da máquina localiza respostas mais rapidamente em dados não estruturados do que um computador 
programável porque o aprendizado de máquina pode trabalhar muito rapidamente com as combinações de fatores 
para encontrar a melhor solução. Computadores programáveis trabalham melhor com dados estruturados localizados 
em bancos de dados.
O aprendizado de máquina resolve problemas de três maneiras:
- Aprendizado Supervisionado, quando se oferece à IA exemplos suficientes para fazer previsões precisas. Todos 
os algoritmos do aprendizado supervisionado precisam de dados rotulados. Dados rotulados são dados agrupados 
em amostras marcadas com um ou mais rótulos.
- Aprendizado não supervisionado, uma pessoa alimenta uma máquina com uma grande quantidade de informações, 
faz uma pergunta, e então a máquina deve descobrir como responder sozinha à questão. O aprendizado não 
supervisionado é útil quando você não sabe como classificar os dados.
- Aprendizado por reforço, é um modelo de aprendizado de máquina semelhante ao aprendizado supervisionado, 
mas o algoritmo não é treinado usando dados de amostra, este aprende à medida que avança usando tentativa e 
erro, dá ao seu agente reforço positivo por realizar a coisa "certa" e reforço negativo por realizar as coisas
 "erradas".
